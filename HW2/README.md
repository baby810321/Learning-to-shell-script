# HW2
利用 getopts 做出支援 argument parser 版的 calc.sh

## Installation
下載 calc.sh 至 Linux 環境

## Usage
./calc.sh -o add -a 4 -b 2
a + b = 6
./calc.sh -o sub -a 4 -b 2
a + b = 2
./calc.sh -o mul -a 4 -b 2
a * b = 8
./calc.sh -o div -a 4 -b 2
a / b = 2
